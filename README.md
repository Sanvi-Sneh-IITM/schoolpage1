# School Hub Website

A simple, responsive single-page website for a school, built with Tailwind CSS.

## Features

*   **Responsive Design:** Adapts to various screen sizes (mobile, tablet, desktop).
*   **Themed Background:** A pleasant, school-appropriate colored background.
*   **Simple Navigation:** Basic header navigation.
*   **Informative Sections:** Areas for announcements, academic programs, student life, and admissions.
*   **Single File:** All HTML and styling (via Tailwind CDN) are contained in `index.html`.

## Technologies Used

*   **HTML5:** For the page structure.
*   **Tailwind CSS (CDN):** For utility-first styling and responsiveness.

## Setup and Usage

This project is a single HTML file. To view it:

1.  **Save the file:** Save the content of `index.html` into a file named `index.html` on your computer.
2.  **Open in Browser:** Double-click the `index.html` file, or open it with your preferred web browser.

That's it! No build steps or complex setup required.
